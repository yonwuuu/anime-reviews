# anime-reviews
Суть проекта создание аниме сайта с рецензиями на различные аниме. Возможность ознакомится с понравившемся на сайте аниме детальнее. Увидеть жанр, главных героев, дату выпуска и другую интереснующию информацию по поводу двнного произведения. 
Так же из фунционала будет доступная система оценивания по 5 бальной системе каждый пользователь иожет поставить оценку данному аниме, тем самым будет отображаться рейтинг данного тайтла, и на основе этого будет возможность посмотреть какой тайтл на сайте среди пользователей пользуется  бщльшим спросом, а какой меньшим. Тфк же скорее всего будет добавлен функционал с написанием анонимных коментариев под разбором каждого аниме и более детальная возможность выражения мнения для пользователя.
 Языки програмирования которыми будет проделана работа это.

 JavaScript
 CSS
